# Cook with Puppy
Recipe app which enables a user to create his own recipes or search for new ones using API. There is also a possibility to add a recipe to Favourites.

## Used Tech
* Kotlin
* MVVM
* Room
* LiveData
* List adapter
* API - https://rapidapi.com/brianiswu/api/recipe-puppy

Recipe can be changed and deleted by user. It contains title, list of ingredients and (optionally) link to the website.
